# crazy_ball

### Quick start guide

Clone the repository, then run in the clonned directory the next commands:

```
mkdir build
cd build
cmake ..
cmake --build .
./main
```

**Note:** For a **not windows os**, maybe it's nessesary to set `SFML_STATIC_LIBRARIES` to `FALSE` in the **CMakeLists.txt**.

### Controls

`←` - **move to the left**

`→` - **move to the right**

`Space` - **jump**
